# WOW Bots — Universal Telegram Business Bot

## Структура файлів
```
wow_bots/
├── bot.py          # Основний код бота
├── config.json     # Конфіг клієнта (редагуй це!)
├── index.html      # Лендінг сторінка
└── README.md
```

## Налаштування для нового клієнта

1. Скопіюй `config.json`
2. Замінив `name`, `owner_chat_id`, `greeting`
3. Заповни `categories` → `subcategories` → `items`
4. Додай `address` і `faq`
5. Встанови `BOT_TOKEN` і запусти

## Запуск
```bash
pip install python-telegram-bot
BOT_TOKEN=твій_токен python bot.py
```
